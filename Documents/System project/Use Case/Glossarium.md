# Глоссарий

|Термин|Определение|
|-|-|
|Пользователь|Человек, взаимодействующий с интерфейсом приложения для создания штрихкодов.|
|Тип штрихкода|Стандарт графического представления данных (например, QR-код, EAN-13, Code 128), определяющий структуру и внешний вид кода.|
|Тип информации|Категория данных, которые будут зашифрованы в коде (например, URL-адрес, простой текст, контактные данные, номер телефона).|
|Валидация данных|Проверка введенной пользователем информации на соответствие правилам выбранного типа штрихкода (например, только цифры для EAN-13).|
|Генерация|Процесс преобразования введенных текстовых данных в графическое изображение штрихкода программными средствами.|
|Предпросмотр|Отображение сформированного изображения штрихкода на экране приложения перед его сохранением.|
|Экспорт (Скачивание)|Сохранение сгенерированного изображения штрихкода в память устройства пользователя в виде файла (в формате PNG или SVG).|
