# 2 Варианты использования

<a name="enter\_in\_the\_app"/>

## 2.1 Сгенерировать штрихкод

Описание. Вариант использования «Сгенерировать штрихкод» позволяет пользователю создать графическое изображение кода на основе выбранных параметров и введенных данных.

Предусловия. Приложение запущено и готово к работе.

Основной поток.

1. Приложение отображает главную форму с выбором типов штрихкода;
2. Выполняется вариант использования «Выбрать тип штрихкода»;
3. Выполняется вариант использования «Ввести данные»;
4. Пользователь нажимает кнопку «Сгенерировать»;
5. Приложение выполняет валидацию данных. Если данные корректны, система отрисовывает штрихкод на экране;
6. Вариант использования завершается.

Постусловия. Пользователю доступен вариант использования «Скачать штрихкод».

<a name="sign\_in"/>

## 2.2 Выбрать тип штрихкода

Описание. Позволяет пользователю определить стандарт кодирования.

Основной поток.

1. Пользователь выбирает один из доступных стандартов: EAN-8, EAN-13, Data Matrix или QR-код;
2. Пользователь выбирает нужный тип штрихкода;
3. Приложение фиксирует выбор и подгружает соответствующие алгоритмы генерации;
4. Вариант использования завершается.

<a name="view\_tasks\_list"/>

## 2.3 Выбрать тип данных

Описание. Позволяет пользователю указать характер информации (текст, ссылка, число), которую нужно зашифровать.

Основной поток.  

1. Приложение отображает перечень форматов данных, доступных для выбранного типа штрихкода;
2. Пользователь выбирает тип данных;
3. Приложение адаптирует поле ввода под выбранный формат;
4. Вариант использования завершается.

<a name="download\\\_data"/>

## 2\.4 Ввести данные

Описание. Позволяет пользователю ввести текстовое или числовое значение для кодирования.

Основной поток.

1. Приложение активирует поле ввода, соответствующее выбранному типу штрихкода;
2. Пользователь вводит информацию;
3. Вариант использования завершается.

<a name="view\_important\_tasks\_list"/>

## 2\.5 Скачать штрихкод

Описание. Позволяет сохранить результат в виде файла.

Предусловия. Штрихкод успешно сгенерирован и отображен на экране.  
Основной поток.

1. Пользователь нажимает кнопку «Скачать»;
2. Приложение формирует файл изображения;
3. Приложение сохраняет файл в память устройства;
4. Вариант использования завершается.
